Overview

Darkone is a modern, developer-centric, and highly customizable React dashboard template. Built on Bootstrap 5, SCSS, it features a fully responsive layout and an intuitive interface. With a vast library of components, widgets, and UI elements, Darkone accelerates project development, saving valuable time and resources. Perfect for SaaS interfaces, admin dashboards, or any other web application, Darkone streamlines your workflow.

Setup Guide

Prerequisites
  • Node.js: Install Node.js to manage dependencies.
  • Yarn: Make sure to have the installed & running in your computer.
Installation
  • Step 1: Ensure all prerequisites are installed and running.
  • Step 2: Use npm install or yarn to install project dependencies.

After installation, use these commands:

Command Description
yarn Generate node-module package folder
yarn dev Runs the project locally, starts the development server.
yarn build It bundles with production mode. Your app is now ready to be deployed.
Command Description
npm i This would install all the required dependencies in the node_modules folder.
npm run dev Runs the project locally, starts the development server.
npm run build It bundles with production mode. Your app is now ready to be deployed.
Command Description
bun i This would install all the required dependencies in the node_modules folder.
bun dev Runs the project locally, starts the development server.
bun run build It bundles with production mode. Your app is now ready to be deployed.
